串口3 PID调参 速查卡
=====================

接线: PB10(TX) + PB11(RX) + GND, 波特率 115200

=== PID 调节 ===
P60 回车          → 两电机 P=60
I40 回车          → 两电机 I=40
D5 回车           → 两电机 D=5
PA80 回车         → 只调电机A的 P=80
IA30 回车         → 只调电机A的 I=30
DA3 回车          → 只调电机A的 D=3
PB70 回车         → 只调电机B的 P=70
IB20 回车         → 只调电机B的 I=20
DB5 回车          → 只调电机B的 D=5

=== 巡线速度 ===
S50 回车          → 直道=50, 其余4档自动按比例缩放

=== 巡线算法 ===
G12 回车          → 转向增益(8~18): 越大转弯越猛
C0.6 回车         → 弯道减速(0~1): 越大弯道越慢

=== 巡线转向 PD ===
KP2.5 回车        → 转向比例(0.5~8): 越大转弯越猛
KD3.5 回车        → 转向微分(1~10): 越大越稳但越"肉"

=== 查询 ===
? 回车            → 打印当前全部参数(PID/速度/编码器/巡线状态)
HELP 回车         → 打印命令列表

=== 典型调参流程 ===
S50 回车          → 设定速度基准, 回显5档速度值
? 回车            → 查看当前全部参数
P80 回车          → 加大 P, 串口1波形看响应
IA20 回车         → 调电机A的 I
DB3 回车          → 单独给电机B加 D
G14 回车          → 转弯不够猛就加大转向增益
? 回车            → 确认改动后的值
