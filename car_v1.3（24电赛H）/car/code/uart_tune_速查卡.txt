串口3 PID调参 速查卡
=====================

接线: PB10(TX) + PB11(RX) + GND, 波特率 115200

=== PID 调节 (速度环, 增量式 DELTA_PID) ===
P<值> 回车        → 两电机 P (当前=200)
I<值> 回车        → 两电机 I (当前=88)
D<值> 回车        → 两电机 D (当前=15)
PA<值> 回车       → 只调电机A P
IA<值> 回车       → 只调电机A I
DA<值> 回车       → 只调电机A D
PB<值> 回车       → 只调电机B P
IB<值> 回车       → 只调电机B I
DB<值> 回车       → 只调电机B D

=== 巡线速度 ===
S<值> 回车        → 直道速度, 其余2档按比例自动缩放 (当前=30)
                    如: S20 → FAST=20 SLOW=11 MIN=1

=== 巡线转向 PD ===
KP<值> 回车       → 转向P: 越大转弯越猛 (当前=3.0, 建议2~5)
KD<值> 回车       → 转向D: 越大越稳但越"肉"  (当前=4.0, 建议2~6)

=== 查询 ===
? 回车            → 打印全部参数 (PID/速度/转向PD/编码器/状态)
HELP 回车         → 打印命令列表 (含当前值)

=== 典型调参流程 ===
? 回车            → 先看当前全部参数
S30 回车          → 设定直道基准速度, 自动缩放其余档位
KP4 回车          → 转弯不够猛 → 加大转向P
KD5 回车          → 车身左右摆 → 加大转向D
P250 回车         → 速度跟不上 → 加大电机P
? 回车            → 确认改动后的值
